sift_status_t sift_device_read_reg32(sift_device_t dev, uint32_t *value, sift_aperture_t ap, uint64_t reg)

reads a 32-bit MMIO register

Parameters:

  • dev – device handle

  • ap – aperture

  • reg – address of register to read

  • value – pointer to memory to populate with the value read

Returns:

SIFT_STATUS_SUCCESS on success
